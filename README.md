Clone repository:
```
$ git clone https://github.com/AleksandrSekker/sheeptest.git
```
or
```
$ git clone git@github.com:AleksandrSekker/sheeptest.git
```

Install project dependencies:
```
$ pnpm install
```

Run the project:
```
$ pnpm run dev 
```
